Circunferencia Elipse Parábola Hipérbola Herramientas Referencias

Circunferencia

Definición (Español): Es el conjunto de puntos que equidistan de un punto fijo llamado centro. Se usa ampliamente en geometría, diseño y construcción.

Definition (English): A circle is the set of all points in a plane that are equidistant from a fixed point called the center. It is widely used in geometry, design, and construction.

Ecuación: (x - h)² + (y - k)² = r²

Ejemplo: Si el centro es (0,0) y el radio es 3 → x² + y² = 9.

Elipse

Definición (Español): Es el lugar geométrico de los puntos cuya suma de distancias a dos puntos fijos llamados focos es constante. Se aplica en astronomía y física.

Definition (English): An ellipse is the set of all points for which the sum of the distances to two fixed points (foci) is constant. It is applied in astronomy and physics.

Ecuación: (x² / a²) + (y² / b²) = 1

Ejemplo: Para a=5 y b=3 → (x²/25) + (y²/9) = 1.

Parábola

Definición (Español): Es el conjunto de puntos equidistantes de un punto fijo (foco) y una recta fija (directriz). Tiene aplicaciones en óptica y trayectorias.

Definition (English): A parabola is the set of points equidistant from a fixed point (focus) and a fixed line (directrix). It has applications in optics and trajectories.

Ecuación: y² = 4ax o x² = 4ay

Ejemplo: Para x² = 8y, el foco está en (0,2).

Hipérbola

Definición (Español): Es el lugar geométrico de los puntos cuya diferencia de distancias a dos puntos fijos llamados focos es constante. Es importante en navegación y telecomunicaciones.

Definition (English): A hyperbola is the set of all points in a plane where the difference of the distances to two fixed points (foci) is constant. It is important in navigation and telecommunications.

Ecuación: (x² / a²) - (y² / b²) = 1

Ejemplo: Para a=4 y b=3 → (x²/16) - (y²/9) = 1.
